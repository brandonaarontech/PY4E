# Python for Everybody (PY4E)

This repo contains my exercise solutions to the PY4E course by Dr. Charles Severance 

The purpose of this repo is to reinforce core Python programming concepts 

## Skills Demonstrated

- Python syntax and control flow
- String parsing and manipulation
- File input and processing
- Error handling with try/except
- Basic data extraction from structured text

## Repository Structure

- CH_01 – Introduction  
- CH_02 – Variables and Expressions  
- CH_03 – Conditional Execution  
- CH_04 – Functions  
- CH_05 – Loops and Iteration  
- CH_06 – Strings  
- CH_07 – Files and Data Processing  

Each chapter directory contains the required exercises from each section
